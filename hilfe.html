Erste Schritte

1

Airtable-Konfiguration

Verbinden Sie die Anwendung mit Ihrer Airtable-Datenbank, um auf Kundendaten zuzugreifen.

Tipp: Sie finden Ihren API-Key und Base-ID in den Einstellungen Ihres Airtable-Accounts.

2

Kunden auswählen

Wählen Sie einen Kunden aus Ihrer Datenbank aus, für den Sie eine 37.3-Prüfung durchführen möchten.

  • Nutzen Sie die Suchfunktion, um Kunden schnell zu finden
  • Filtern Sie nach Region, Status oder Pflegegrad
  • Klicken Sie auf "Neue Prüfung", um zu starten
3

Formular ausfüllen

Füllen Sie das 37.3-Formular schrittweise aus. Die Anwendung füllt automatisch bekannte Kundendaten ein.

  • Überprüfen Sie die vorausgefüllten Daten
  • Ergänzen Sie fehlende Informationen
  • Die PDF-Vorschau zeigt Ihnen den aktuellen Stand
4

PDF generieren

Generieren Sie das ausgefüllte PDF-Dokument und speichern oder drucken Sie es.

Hinweis: Das PDF wird automatisch mit aktuellem Datum und Kundendaten benannt.

Häufig gestellte Fragen

Wie verbinde ich die Anwendung mit Airtable?

Gehen Sie zu den Einstellungen und geben Sie Ihren Airtable API-Key und Base-ID ein. Sie finden diese Informationen in Ihrem Airtable-Account unter "Account" → "API".

  1. API-Key: Wird in Airtable unter Account → API generiert
  2. Base-ID: Finden Sie in der URL Ihrer Airtable-Base
  3. Tabelle: Der Name Ihrer Kundentabelle (z.B. "Kunden")

Welche Daten werden automatisch aus Airtable geladen?

Folgende Kundendaten werden automatisch geladen und in das Formular eingefügt:

  • Name, Vorname, Nachname
  • Geburtsdatum
  • Adresse (Straße, PLZ, Ort)
  • Versicherungsnummer
  • Pflegegrad
  • Kontaktdaten (Telefon, Email)

Kann ich die PDF-Vorlage anpassen?

Ja, Sie können eigene PDF-Vorlagen hochladen. Gehen Sie dazu zu den Einstellungen → PDF-Vorlagen und laden Sie Ihre angepasste Vorlage hoch.

Stellen Sie sicher, dass Ihre Vorlage die korrekten Formularfelder enthält, damit die Daten richtig zugeordnet werden.

Werden meine Daten automatisch gespeichert?

Ja, das Formular wird alle 30 Sekunden automatisch im lokalen Speicher Ihres Browsers gespeichert. Sie können die Einstellungen ändern, um das automatische Speichern zu deaktivieren.

Wichtig: Der automatische Speicher erfolgt nur lokal in Ihrem Browser, nicht in Airtable.

Wie kann ich bestehende Prüfungen bearbeiten?

Aktuell werden Prüfungen als PDF-Dokumente generiert und sind nicht direkt in der Anwendung bearbeitbar. Um eine bestehende Prüfung zu aktualisieren:

  1. Wählen Sie den gleichen Kunden erneut aus
  2. Die bekannten Daten werden wieder eingefügt
  3. Aktualisieren Sie die notwendigen Felder
  4. Generieren Sie ein neues PDF

Welche Browser werden unterstützt?

Die Anwendung wurde für moderne Browser optimiert und unterstützt:

  • Google Chrome (empfohlen)
  • Mozilla Firefox
  • Microsoft Edge
  • Safari (ab Version 14)

Stellen Sie sicher, dass JavaScript aktiviert ist und Cookies zugelassen sind.
